# RPS Game

A fun Rock Paper Scissors game for the terminal!

## Installation

### Windows/Mac/Linux
```bash
pip install git+https://github.com/SohanurRahmanSohan43/rps_game.git
rps
```

### Android (Termux)
1. Install [Termux from F-Droid](https://f-droid.org/packages/com.termux/)
2. Run:
```bash
pkg update && pkg upgrade
pkg install python git
git clone https://github.com/SohanurRahmanSohan43/rps_game.git
cd rps_game
pip install .
rps
```

### From Source
```bash
git clone https://github.com/SohanurRahmanSohan43/rps_game.git
cd rps_game
pip install .
rps
```

## Usage
Just type `rps` in your terminal and enjoy!

## Controls
- `r` - Rock
- `p` - Paper
- `s` - Scissors
- `Ctrl+C` - Exit and see scoreboard

## Uninstallation
```bash
pip uninstall rps-game-cli
```
